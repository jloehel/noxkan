# Security Policy

Noxkan is a fork of Faraday:
https://github.com/infobyte/faraday

Security issues should be reported via GitHub Security Advisories:

https://github.com/jloehel/noxkan/security/advisories

Please do not disclose vulnerabilities publicly before coordinated disclosure.

If an issue affects unmodified upstream components, it may also be reported to the Faraday maintainers:
https://github.com/infobyte/faraday
