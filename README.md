# Noxkan

Noxkan is a fork of Faraday:
https://github.com/infobyte/faraday

Faraday is an open source vulnerability management platform developed by Infobyte.  
Noxkan builds directly on that codebase.

All original copyright and license notices are preserved.  
This project remains licensed under GPL-3.0.

## Scope

At this stage, Noxkan reflects the current upstream functionality with project-level adjustments.  
Functional changes and extensions will be introduced incrementally and documented per release.
