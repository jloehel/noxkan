# Code of Conduct

## Our Standards

We expect all participants to:

- Be respectful and constructive
- Focus on technical discussion
- Avoid harassment or discrimination
- Accept reasonable criticism

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, and discussions.

## Enforcement

Project maintainers are responsible for enforcing this policy.  
Unacceptable behavior may result in warnings, temporary restrictions, or permanent bans.

## Reporting

Report conduct concerns via GitHub or direct contact with the maintainers.
